# CNNFlexabilityExperiment
Kaggle Notebook project that aims to explore the flexability of CNNs by training one on Brain Tumor object detection and seeing if it can detect tumors in other organs.

# Project Goal
The purpose of this project is to experiment with the flexability of CNNs within the context of detecting objects that are similar to, but not exactly the same as what the CNN has been trained to detect. In this case, we will be training a CNN to detect brain tumors exclusively and then show it images of tumors in other organs to see if it can still detect them with relative accuracy.
